◆ getBusyLevel()

int Animation::getBusyLevel ( )

ビジーレベルを取得する

戻り値
現在のアニメーションのビジーレベル

ビジーレベルとは

アニメーション（アクション）ごとの優先度を管理するための値。

ビジーレベルの高いアニメーション再生中は、よりビジーレベルの低いアニメーションは割り込んで再生できない。

（例：ジャンプ中（ビジーレベル１）は、左右に移動しても歩行アニメーション（ビジーレベル０）を再生しない）
